# CLAUDE.md

## Repository Purpose

This repository contains the AI Project Operating System (AI-POS) Standard.

The objective is to build a reusable, AI-agnostic framework for software development, automation, documentation, research, and business workflows.

## General Instructions

Before making changes:

1. Read the relevant documentation.
2. Preserve existing architecture.
3. Avoid duplicate work.
4. Update documentation when implementation changes.
5. Record important decisions.
6. Prefer reusable solutions over one-off implementations.

## Documentation Rules

- Keep Markdown files well organized.
- Use descriptive headings.
- Keep examples current.
- Avoid contradictory documentation.

## Coding Rules

- Prefer readability over cleverness.
- Keep functions focused.
- Comment only where necessary.
- Preserve consistency with the existing codebase.

## AI Collaboration

Assume multiple AI assistants may contribute.

Leave work in a state where another AI can continue immediately without requiring additional explanation.

## Quality Standard

Every deliverable should be:

- Complete
- Consistent
- Maintainable
- Reusable
- Well documented

## End Goal

Build a production-grade AI Project Operating System that can be reused across future projects.
